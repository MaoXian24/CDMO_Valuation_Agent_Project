# Report Style Guide (CDMO)

Tone: concise, analyst-style, structured paragraphs.

Structure:
1) Industry overview (market drivers, pricing, capacity, regulation).
2) Company positioning (capacity, client mix, pipeline, geography).
3) Financial performance (growth, margins, cash flow).
4) Valuation summary (method, key drivers, sensitivity).
5) Risks and limitations.

Style Notes:
- Avoid marketing language.
- Use numbers and comparisons.
- Cite chart references by caption.
