# Group Coursework Requirements Checklist (Condensed)

- Select 1 industry from the teaching team list.
- Build an AI valuation agent that supports end-to-end equity research workflow.
- Include automated analysis and human-in-the-loop decision points.

## Required Workflow Stages
- Company input (ticker, segment, etc.).
- Data collection (target company and peers).
- Historical analysis (ratios, profitability, efficiency, leverage, liquidity).
- Interactive assumptions (growth, margins, discount rate, terminal growth).
- Valuation model (DCF or industry-appropriate method).
- Output generation (analysis text, charts, tables, commentary).

2) 3-5 minute product demonstration video.
3) Prompt and orchestration documentation (roles, prompts, workflow, tools, interaction design, exported workflow file).
4) Sample output pack for at least one company.
5) User guide (target user, inputs, outputs, limitations).

## Evaluation Emphasis
- Industry fit and usefulness.
- Prompt quality and style control.

- Output quality (analysis, valuation, charts, tables).
- Demo clarity and user guide usability.
