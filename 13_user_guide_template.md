# User Guide Template

## Product Name
CDMO Valuation Agent

## Target Users
Equity research analysts covering the CDMO segment.

## What the Tool Does
- Retrieves and cleans company data
- Computes key financial ratios
- Runs DCF valuation with user-approved assumptions
- Produces analyst-style report sections and charts
- Optionally exports the finished output package if the platform supports it

## Required Inputs
- Company name or ticker
- Industry segment
- Optional peer list
- Optional assumption overrides

## Output Interpretation
- Ratios show historical operating performance
- DCF outputs provide intrinsic value estimates
- Charts summarize trends and comparisons

## Limitations
- Data quality depends on availability and consistency
- Assumptions can materially affect valuation outcomes
- Some company-specific qualitative factors may be missing
- Export/download behavior may vary by platform and should not change the valuation result
